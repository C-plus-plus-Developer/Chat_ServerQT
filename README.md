# Chat_ServerQT
Сервер чата, интерфейс сделан в QT на С++
Ссылка на клиент: https://github.com/C-plus-plus-Developer/Chat_KlientQT.git
Что сделать чтбы все заработало:
1. Создать базу данных в MySQL и в 33 строке файла Chat.h ввести данные чтобы программа смогла найти ее. Никаких таблиц создавать не надо, программа все сделает сама.
2. Запустить проект в QT Creator. Перед запуском на всякий случай запустите бд с помощью команды sudo systemctl start mysql
Если захотите изменить порт, то он находится в Chat.h после заголовочных файлов, но нужно также изменить и клиент, его порт находится также в Chat.h после заголовочных файлов.
